# 🚀 야구 게임 기능 목록

- [X] 1부터 9까지 서로 다른 수 3자리를 입력한다.
  - [X] [예외] 1부터 9까지의 숫자가 아닌 경우
  - [X] [예외] 3자리가 아닌 경우
  - [X] [예외] 입력을 하지 않은 경우
  - [X] [예외] 문자를 입력한 경우
  - [X] [예외] 중복되는 경우
- [X] 컴퓨터는 3자리 수를 생성한다.
- [ ] 컴퓨터와 플레이어를 비교한다.
  - [ ] 같은 수 같은 자리인 경우 스트라크
  - [ ] 같은 수 다른 자리인 경우 볼
  - [ ] 같은 수가 없는 경우 낫싱
- [X] 게임 재시작 여부 입력한다.
  - [X] [예외] 1또는 2를 입력하지 않은 경우
  - [X] [예외] 문자를 입력한 경우
  - [X] [예외] 1글자가 아닌 경우
  - [X] [예외] 입력을 하지 않은 경우
